# Supported Versions

0XB20 Laboratory is maintained as a fast-moving public experiment.

| Version | Status | Notes |
| --- | --- | --- |
| `main` | Supported | Current public release branch. |
| Deployed production site | Supported | Mirrors the latest stable `main` deployment. |
| Historical commits | Unsupported | Kept for transparency only. |

Security fixes target `main` first.
